This project uses a random number generator to generate a head or a tail. It counts the heads and tails and assigns them to an array.

The heads are collected in one array and the tails in another. They are then pushed to create an array of 100 pairs.

This is repeated 100,000 times and the result of the maximum is stored in a new array.
